Premiere Pro Tips

10/12/2017

Shortcuts

  • V - Selection tool
  • A - Track select forward tool
  • B - Ripple edit tool cut from one side and add from another
  • N - Rollling edit tool - edits clips length by using the tails of adjescent clip
  • R - Rate stretch tool - speed up or slow down the video
  • C - Razor tool
  • Y - Slip tool - moving with the clip below the cliped area (the position of the clip is not changing, just the content)
  • U - Slide tool - keeps the length of the dragged clip and shortens the length of the clip next to that dragged clip
  • pen tool, hand tool, text tool

Nested sequences

  • Nested sequences
  • Select clips -> nest
  • you can edit clips individually and you have to make nested sequence clip long enough
  • or sequence -> make subsequence

Multicam editing

  • Multicam editing
  • 1) Import -> select clips create multicamera sequence
  • 2) Click it -> new sequence from the clip, hit OK
  • 3) hit + below video preview -> multicam btn, click it
  • 4) Play the video and click on video 1, 2.. Where you want to switch (or use 1, 2 keys) (The cuts will be created automatically)

Tracking

  • 1) Import your video
  • 2) apply mosaic filter on it
  • 3) create elipse mask over person face in effect control panel
  • 4) move on the begining of the clip and hit the play button on the mask in the effect control panel to track its movement
